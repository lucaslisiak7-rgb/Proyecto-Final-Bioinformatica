# Proyecto-Final-Bioinformatica
Este proyecto corresponde al trabajo final de la materia Bioinformática (2025), donde se aplican técnicas de aprendizaje automático para el reconocimiento de actividades humanas a partir de datos obtenidos por los sensores inerciales de un smartphone (acelerómetro y giroscopio).
El análisis incluye etapas de preprocesamiento, reducción de dimensionalidad (PCA) y el uso de modelos de aprendizaje supervisado y no supervisado, entre ellos Árboles de Decisión, Random Forest, SVM, Redes Neuronales (MLP), K-Means y Clustering Jerárquico.
El objetivo es comparar el rendimiento de distintos algoritmos y comprender cómo las señales biométricas pueden emplearse para el monitoreo, la rehabilitación y el análisis del movimiento humano.
El notebook está orientado a estudiantes o profesionales que deseen explorar el uso de Python y scikit-learn en el análisis de datos biomédicos.
